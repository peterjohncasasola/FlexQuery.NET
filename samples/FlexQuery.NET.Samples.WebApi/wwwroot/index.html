📝 Copy-Paste Controller

This is the entire backend for the AG Grid SSRM demo — 3 lines of FlexQuery

📄 AgGridController.cs 
public async Task<IActionResult> GetCustomers(
    [FromBody] AgGridRequest request)
{
    // 1. Parse AG Grid SSRM request → QueryOptions
    var options = request.ToQueryOptions();
    options.IncludeCount = true;

    // 2. Execute against EF Core (or Dapper!)
    var result = await _db.Customers
        .AsNoTracking()
        .FlexQueryAsync<Customer>(options);

    // 3. Return AG Grid-compatible response
    return Ok(result.ToAgGridServerSideResponse(request));
}

📈 Features at a Glance

🔍 Filtering

DSL or JQL syntax with operators: eq, neq, contains, startswith, endswith, gt, gte, lt, lte. AND/OR group logic.

?filter=Status:eq:Active

📄 Sorting

Multi-column sort with asc/desc direction. Works across all providers.

?sort=LastName:asc,City:desc

👪 Paging

Page-based pagination with total count. Server-side skip/take.

?page=1&pageSize=20

👓 Projection

Select only the fields you need. Reduces bandwidth and query cost.

?select=Id,FirstName,Email

🔗 Include

Eager-load navigation properties with optional filtered includes.

?include=Orders

📈 Aggregation

Count, sum, average, min, max with GROUP BY and HAVING support.

?groupBy=Status&aggregate=count:Id:count
